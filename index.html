🦞 OpenClaw 龍蝦安裝教學

從零開始，10 分鐘擁有你的 AI 管家

什麼是 OpenClaw？

OpenClaw（龍蝦）是開源 AI Agent 框架，讓 AI 擁有「手」和「長期記憶」。
簡單說：ChatGPT 是「會回答的大腦」，OpenClaw 是「會動手做事的管家」。

🖥️ 操作電腦 📱 30+ 通訊平台 🧩 5,700+ 技能 ⏰ 24/7 常駐 💾 RAM 2GB 即可

安裝步驟

1

一鍵安裝

macOS / Linux 打開終端機，貼上以下指令：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows 用 PowerShell（系統管理員）：

iwr -useb https://openclaw.ai/install.ps1 | iex
2

初始化設定

執行設定向導，依序選擇 AI 模型、通訊平台、技能模組：

openclaw onboard

向導會引導你完成：啟動模式 → AI 供應商 → 聊天工具 → 埠口 → 技能 → API Key

3

接上 Telegram

先到 Telegram 找 @BotFather 建立 Bot 拿 Token，然後設定：

openclaw config set telegram.token "你的Bot Token"
4

啟動！

一切就緒，啟動 OpenClaw：

openclaw start

瀏覽器會自動開啟 http://127.0.0.1:18789/chat，在 Telegram 對 Bot 發訊息測試即可。

5

安裝技能

OpenClaw 的技能就像 App，按需安裝：

openclaw skill search 關鍵字 openclaw skill install 技能名稱 openclaw skill list

常用：web-search（搜尋）、file-manager（檔案）、calendar（行事曆）、email-agent（郵件）

6

常駐運行（24/7）

用 pm2 讓 OpenClaw 不斷線：

npm i -g pm2 pm2 start "openclaw start" --name lobster pm2 save && pm2 startup

常見問題

安裝失敗？

確認 Node.js 版本 ≥ 22，執行 node -v 檢查

Bot 沒回應？

確認 Token 正確且 OpenClaw 正在運行中

記憶體不足？

最低需求 2GB RAM，關掉不必要的程式

想換 AI 模型？

執行 openclaw config set model.provider 供應商名

技能裝不上？

執行 openclaw skill update 更新清單後重試

怎麼接 LINE？

onboard 時選擇 LINE，或用 openclaw config 設定
